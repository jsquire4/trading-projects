# trading-projects

Mono-repo for trading infrastructure projects:

- **Equinox** — Cross-venue prediction market aggregation and routing simulation
- **CapMan AI** — Gamified scenario-based trading training with MTSS agent
- **Meridian** — Binary stock outcome markets (0DTE) on blockchain
- **Volatility Framework** — Options volatility data framework (IV, RV, Greeks, order flow)

## Stack

Python, Solidity, TypeScript

## Status

In development.
